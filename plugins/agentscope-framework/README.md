# agentscope-framework

Plugin cung cấp skills để vibe code AI Agent applications sử dụng [AgentScope](https://github.com/agentscope-ai/agentscope) — production-ready Python agent framework.

## Skills

| Skill | Mô tả |
|---|---|
| `agentscope-agent-architect` | Design & scaffold ReActAgent, multi-agent systems |
| `agentscope-tool-builder` | Tạo tools, toolkit, và tích hợp MCP |
| `agentscope-session-memory` | Quản lý memory (InMemory, LongTerm, ReMe) |
| `agentscope-pipeline` | MsgHub, sequential & fanout pipeline orchestration |
| `agentscope-deploy` | Tracing với OpenTelemetry, Docker, Kubernetes, Serverless |

## Cài đặt

```
/plugin install agentscope-framework@ecb-claude-market
```

## Yêu cầu

- Python 3.10+
- `pip install agentscope`
